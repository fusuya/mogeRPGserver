- 第2回大会ルール  
参加者が5名だったのでグループに分けせずにそれぞれ五回実行した時の獲得ポイントを競った。  

- 獲得ポイント算出方法（暫定）  
ポイントは、到達階数 + ボス討伐ポイントとする。  
ボス討伐ポイントは100階のボスを倒すと５０ポイント、５０階の中ボスを倒すと２０ポイントとする。  
例  
・中ボスを倒さずにダンジョンクリアした場合  
到達階数 100 + ボス討伐ポイント 50 = 150ポイント  
・中ボスを倒したけど６０回で死亡した場合  
到達階数 60 + 中ボス討伐ポイント 20 = 80ポイント
