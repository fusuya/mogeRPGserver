# 第一回もげRPG大会結果  

- 第一回大会ルール  
参加者が７名だったので２グループに分け(一度に観戦できるのが４つなので)予選を行った。  
予選グループ決めるためにディレイ無しで一回づつAIを走らせる。  
１位３位５位７位がAグループ、２位４位６位がBグループとした。  
予選ルールは５回AIを走らせた時の合計到達階数が多かった上位２名が決勝進出。  
決勝ルールはAIを一回走らせた時の到達階数を競う。  
クリアしたAIが複数出たときはクリアタイムを競う。
